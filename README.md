## 游戏列表

### 五子棋 (Gobang/Gomoku)
纯 Web 实现，单文件 HTML/CSS/JS 游戏。

- **[五子棋游戏入口](games.html)** - 选择版本开始游戏
- **[经典版](gobang.html)** - 传统风格界面
- **[现代版](gomoku.html)** - 现代化界面设计，支持移动端

#### 游戏规则
1. 黑方先行，双方轮流落子
2. 点击棋盘交叉点放置棋子
3. 先连成五子（横、竖、斜任意方向）者获胜
4. 棋盘为 15×15 标准规格
5. 支持悔棋功能
