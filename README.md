## 游戏列表

### 五子棋 (Gobang/Gomoku)
纯 Web 实现，单文件 HTML/CSS/JS 游戏。

- **[五子棋游戏入口](games.html)** - 选择版本开始游戏
- **[经典版](gobang.html)** - 传统风格界面
- **[现代版](gomoku.html)** - 现代化界面设计，支持移动端

#### 游戏规则
1. 黑方先行，双方轮流落子
2. 点击棋盘交叉点放置棋子
3. 先连成五子（横、竖、斜任意方向）者获胜
4. 棋盘为 15×15 标准规格
5. 支持悔棋功能

### 四川麻将
纯 Web 实现，单文件 HTML/CSS/JS 游戏。

- **[四川麻将 - 血战到底](sichuan-mahjong-complete.html)** - 完整规则的四川麻将游戏

#### 游戏规则
1. **缺一门**：开局前选择要缺的花色（万、筒、条），胡牌时手中不能有该花色
2. **血战到底**：一家胡牌后游戏继续，直到牌墙摸完或三家胡牌
3. **吃碰杠胡**：支持完整的吃牌、碰牌、杠牌（明杠、暗杠、补杠）、胡牌
4. **查花猪**：流局时检查玩家是否完成缺门，未完成者为"花猪"
5. **胡牌牌型**：支持平胡、七对、龙七对、清一色、碰碰胡、金钩钓等

#### 游戏特色
- 3 个智能机器人陪玩，具有 AI 决策能力
- 精致的麻将牌设计，使用 Unicode 符号
- 美观的游戏界面，带有动画效果
- 完整的计分系统，支持多种胡牌牌型加分
- 响应式设计，支持移动端
