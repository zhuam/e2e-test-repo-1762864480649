### 梭哈 (Show Hand / Stud Poker)
纯 Web 实现，单文件 HTML/CSS/JS 扑克牌游戏。

- **[梭哈游戏入口](showhand.html)** - 经典扑克游戏，1 人对战 3 位 AI

#### 游戏规则
1. **底注**：每局开始前所有玩家支付底注
2. **暗牌**：每人先发 2 张暗牌（只有自己可见）
3. **第一轮下注**：基于 2 张暗牌进行第一轮下注
4. **明牌**：之后逐轮发 3 张明牌（所有人可见），每发一张进行一次下注
5. **最终比牌**：四轮下注结束后，所有未弃牌的玩家亮牌比大小，牌型最大者赢

#### 操作说明
- **弃牌 (Fold)**：放弃本局，已下注的筹码归入底池
- **过牌 (Check)**：当前无人加注时可选择过牌
- **跟注 (Call)**：匹配当前最高下注额
- **加注 (Raise)**：提高下注额，其他玩家需重新决定
- **全押 (All In)**：将所有筹码一次性押上

#### 手牌大小（从大到小）
1. 🥇 **同花顺** — 同花色且连续的五张牌
2. **四条** — 四张相同点数的牌
3. **葫芦** — 三张相同 + 一对
4. **同花** — 五张同花色
5. **顺子** — 五张连续点数（A2345 为最小顺子）
6. **三条** — 三张相同点数的牌
7. **两对** — 两对相同点数的牌
8. **一对** — 一对相同点数的牌
9. **高牌** — 以上皆无时比最大牌

#### 游戏特色
- 赌场风格绿色牌桌界面
- 3 个智能 AI 对手，基于牌型强度自动决策
- 发牌动画效果
- 完整的下注/加注/全押系统
- 响应式设计，支持移动端
- 每局自动结算，显示所有玩家手牌和牌型

### Camera AI - Android 应用
Android 拍照并调用 AI 模型解读图片的应用程序。

- **[android-camera-app/](android-camera-app/)** - Android 应用项目

#### 功能特色
1. **拍照功能** - 使用 CameraX 实现相机预览和拍照
2. **图库选择** - 支持从相册选择图片
3. **AI 图像分析** - 调用 AI 模型解读图片内容
4. **多模型支持** - 支持 OpenAI GPT-4 Vision、Claude Vision 等

#### 支持的 AI 服务
- OpenAI GPT-4 Vision API
- Anthropic Claude Vision API
- 自定义 API 端点
- 本地 TensorFlow Lite 模型（待实现）

#### 使用方法
1. 使用 Android Studio 打开 `android-camera-app` 项目
2. 在 `ModelInterpreter.java` 中配置您的 API 密钥
3. 编译并运行到 Android 设备或模拟器
4. 授予相机权限后，点击拍照按钮拍摄照片
5. 点击"AI 分析"按钮获取图片解读结果
